Các phép tính

Cũng như trong toán học, JavaScript cũng có các dấu cộng trừ nhân chia để lập trình viên thực hiện phép tính.
Trong JavaScript, có những phép tính như sau:
  • Cộng:
    +
  • Trừ:
    −
  • Nhân:
    *
  • Chia:
    /
  • Lấy số dư:
    %
Sau đây là một số các ví dụ:
  1. // Đáp án: 46
    console . log ( 12 + 34 )

  2. // Đáp án: 63
    console . log ( 92 - 29 )

  3. // Đáp án: 30
    console . log ( 5 * 6 )

  4. // Đáp án: 3
    console . log ( 12 / 4 )

  5. // Đáp án: 2
    console . log ( 11 % 3 )
